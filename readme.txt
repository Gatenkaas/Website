Self Host!

1. Download node.js als je dat al hebt ga naar volgende stap
2.  open command promt met ctrl+R en typ cmd en druk op enter
3. nu komt er een nieuw scherm dat is command promt type npm install http-server -g
4. er komt heel veel op je scherm maak geen zorgen er worden bestanden gedownload voor het hosten
5. op command promt in de map van de website en typ npx http-server
6. als het goed is krijg je een nieuw scherm en linkjes naar je site.
 
 Disclaimer: Deze site werkt alleen op je eiegen netwerk je kan port forwarden of iets zoals ngrok gebruiken om hem te hosten
